Evaluate vendor security postures, classify risks, and generate stakeholder communications with our automated TPRM assessment tool.

Key Features

Automated Risk Assessment

Automatically retrieves SecurityScorecard ratings and evaluates vendors based on multiple risk factors including certifications and public security issues.

Smart Approval Logic

Determines approval status (Approved, Conditionally Approved, or Rejected) based on risk level and compliance with security requirements.

Template Generation

Creates professional, ready-to-send communications for stakeholders with all relevant findings and next steps included.

Compliance Checking

Verifies SOC 2 Type 2, ISO 27001, and other security certifications through automated web scraping.

Public Issue Scanning

Identifies data breaches, vulnerabilities, and negative security news to provide comprehensive risk assessment.

CLI Interface

Easy-to-use command-line interface with both interactive and quick assessment modes for maximum flexibility.

Installation

# Clone the repository
git clone https://github.com/joshuamtm/tprm-helper.git
cd tprm-helper

# Install dependencies
npm install

# Build the project
npm run build

# Run the tool
npm run cli
